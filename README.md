# Spotify Listening Dashboard

A simple dashboard to track your historical listening data from Spotify

Check out my [Streamlit app](https://spotify-listening-app.streamlit.app/).

### How to run it on your own machine

1. Install the requirements

   ```
   $ pip install -r requirements.txt
   ```

2. Run the app

   ```
   $ streamlit run main.py
   ```

3. For demo purposes/if you don't have your spotify data ready, please click the checkbox to work through the app using the default data
